## 特徴

- 100% SCSS
- No JavaScript
  - 別のJavaScriptライブラリと組み合わせて使用することを前提としています
- reboot.css 同梱
- 日本語環境に最適化
- ユニバーサルデザイン
- CSS3 カスタムプロパティによる高度なカスタマイズ
